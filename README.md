# Projecte RA3 i RA4 - Elements multimèdia per a la vostra web

Aquest projecte consisteix en la integració de diversos elements multimèdia a la web del Projecte 1.

## Requisits de Compleció

- **Data d'inici:** 24 de gener de 2025, 09:00
- **Data de venciment:** 16 de març de 2025, 23:59

## Elements a incloure

### 1. Vídeos i àudios incrustats (50%)

- **Controls interactius:** reproduir, pausar, ajustar el volum.
- **Creació d'un anunci:** vídeo promocional per a l'empresa de la web.
  - **Durada:** entre 30 segons i 3 minuts.
  - **Contingut:**
    - Mínim de 3 clips de vídeo diferents.
    - Música de fons.
    - Àudio publicitari (veu en off tipus "vine a coneixe'ns").
    - Efectes: zoom, rotació, escalar, càmera ràpida/lenta, cronòmetre, logotip, text animat, etc.
  - **Publicació:** enllaç al servidor de vídeo.
- **Programari recomanat:** Davinci Resolve, Shotcut, Adobe Premiere, etc.

### 2. Slideshows o carrousels (20%)

- Mostrar imatges en seqüència.
- **Requisits:**
  - Mínim de 3 imatges promocionals editades amb Krita.
  - Aplicació d'efectes i capes per millorar la qualitat visual.

### 3. Mapes interactius (10%)

- Integració de Google Maps o similar.
- **Funcionalitats:** zoom i moviment.

### 4. Infografies interactives (20%)

- Creació amb eines com Infogram, Canva o Figma.
- **Elements clicables o animats.**
- Representació visual clara del negoci, serveis o productes.
